# Ganesha-Analyzer
Ganesha-Analyzer es una herramienta tipo CLI y Web cuya funcionalidad principal es el tomar (logs de Sysmon en Windows) y correlacionar usando Inteligencia Artificial junto con el marco MITRE ATT&amp;CK.

> Proyecto en desarrollo: Este repositorio será actualizado conforme avance mi aprendizaje en temas que aun no son de mi dominio (Parser avanzado, IA, integración multiple, correlación avanzada)


## Visión
Desarrollar una herramienta que permita la correlación inteligente de eventos de seguridad, combinando análisis automatizado, contexto MITRE ATT&CK y capacidad de visualización.

## Objetivos
- Procesar logs de Sysmon y otros orígenes.
- Aplicar modelos de IA para correlación y clasificación de eventos.
- Integrar interfaz CLI y versión Web.
- Generar reportes y alertas basados en tácticas y técnicas MITRE ATT&CK.

## Estado actual
En fase inicial de desarrollo y aprendizaje de:
- Parsing avanzado
- Integración de IA
- Correlación de datos
- Diseño multiplataforma (CLI + Web)

## Roadmap
